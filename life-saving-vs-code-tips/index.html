Life Saving VS Code Tips (For Windows... Idk wtf Mac is)

  1. Alt + Up/Down: Swaps everything in the current line of code with the line above or below it. Keep tapping/holding to move it as far as you want.
  2. Alt + Shift + Up/Down: Copies everything on your cursor's current line and pastes it above or below.
  3. Ctrl + D: Highlighting something and pressing this highlights the next perfect copy of it. Can be pressed many times and it also puts an extra cursor behind each copy, allowing you to edit multiple lines at once!
  4. Ctrl + Left/Right: Moves your cursor to the next space or punction mark. If you also hold shift, you'll highight everything on the way!
  5. Ctrl + Backspace/Del: Deletes all text up to the next space or punctuation mark.
  6. Ctrl + Enter: Creates an empty line of code below your current one; even if your cursor is in the middle of text.
  7. Ctrl + Shift + Enter: Creates an empty line of code above your current one.
  8. Shift + Tab: Any lines highlighted when this is pressed will have all of it's code unindented by two spaces (Opposite of Tab).
  9. GIF is pronounced JIF.
